# 📷 Pasta de Imagens — Anna Victoria Birthday Page

Coloque aqui as fotos que você quer exibir na página.

## Como usar

Nomeie suas fotos assim:

| Arquivo       | Onde aparece                  | Legenda padrão                   |
|---------------|-------------------------------|----------------------------------|
| `foto1.jpg`   | Hero flutuante + Carrossel 1  | "Nosso primeiro encontro"        |
| `foto2.jpg`   | Hero flutuante + Carrossel 2  | "A primeira viagem juntos"       |
| `foto3.jpg`   | Carrossel 3                   | "Um dia qualquer que nunca esqueci" |
| `foto4.jpg`   | Carrossel 4                   | "Momentos que valem uma vida toda" |
| `foto5.jpg`   | Carrossel 5                   | "Te escolhendo todos os dias"    |

## Formatos suportados
- `.jpg` / `.jpeg`
- `.png`
- `.webp`

## Dica
- Use fotos **quadradas** ou próximas de 1:1 para o melhor resultado no estilo polaroid.
- Qualidade recomendada: entre **800x800** e **1200x1200** pixels.
- Se a foto já estiver com outro nome, edite o `index.html` nos trechos `src="images/fotoX.jpg"`.

## Personalização extra

No `index.html` você também pode:
- Trocar o link do vídeo (busque por `SEU_VIDEO_ID`)
- Trocar o link da playlist (busque por `SEU_LINK_AQUI`)
- Editar o texto da carta (seção `section-carta`)
- Mudar os textos dos marcadores do mapa (seção `section-mapa`)
